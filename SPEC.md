# 퍼티스트(Puttist) 퍼팅 연습 앱 서비스 사양서

## 1. 프로젝트 개요

### 1.1 제품명
퍼티스트 퍼팅 연습 앱

### 1.2 비전
"골프의 완성은 퍼팅" - 과학적 원리와 게임화를 통해 효과적인 퍼팅 연습을 제공하는 스마트 퍼팅 트레이닝 플랫폼

### 1.3 목표
- 퍼티스트 퍼팅연습기 사용자들의 연습 효과 극대화
- 지루한 반복 연습을 재미있는 게임으로 전환
- 과학적 데이터 분석을 통한 실력 향상
- 골퍼들 간의 온라인 커뮤니티 형성

### 1.4 대상 사용자
- **주요 타겟**: 퍼티스트 퍼팅연습기 구매 고객
- **부가 타겟**: 모든 레벨의 골퍼 (초보자~프로)

## 2. 제품 연동 사양

### 2.1 퍼티스트 퍼팅연습기 사양
- **물리적 구조**
  - 매트 폭: 20cm
  - 연습 가능 거리: 1.2m ~ 19.9m (볼 하나로 20m까지 연습 가능)
  - 최단 인식 거리: 1.2m (기기가 인식하는 최소 거리)
  - 볼 놓는 지점 표시
  - 퍼터 좌우 이동 거리 가늠 표시
  - 접이식 구조로 편리한 보관

- **디지털 기능**
  - 거리 측정 기술 (핵심 기술)
  - 성공/실패 판정: 지정 거리를 통과한 후 50cm 이내에 멈추면 성공
    - 예시: 5m 목표 시 → 5.0m~5.5m 범위 내 성공
    - 목표 거리 미달 시 실패
    - 목표 거리 + 50cm 초과 시 실패
  - LED 디스플레이
  - 성공 시 불빛 깜박임
  - ESP-32 기반 (블루투스 연동 개발 예정)

- **게임 모드** (모든 모드 동일한 성공 판정 규칙 적용)
  1. **일반 모드**: 1.2m ~ 19.9m 자유 연습
  2. **3-6-9 게임**: 3m, 6m, 9m 순차 도전 (각 거리 3번 기회)
  3. **7-up**: 1m부터 7m까지 1m씩 증가
  4. **7-down**: 7m부터 1m까지 1m씩 감소
  5. **7-random**: 1-7m 랜덤
  6. **9-random**: 1-9m 랜덤

### 2.2 연동 전략 (단계별)
1. **Phase 1 (MVP)**: 수동 데이터 입력
2. **Phase 2**: ESP-32 블루투스 자동 연동
3. **Phase 3**: 하이브리드 모드 (기기 버전별 지원)

## 3. 핵심 기능

### 3.1 데이터 기록 및 관리
#### 3.1.1 수동 입력 모드 (Phase 1)
- 거리별 성공/실패 기록
- 게임 모드별 점수 기록
- 날짜/시간별 연습 기록
- 간편한 입력 UI (원터치 입력)

#### 3.1.2 자동 연동 모드 (Phase 2)
- 블루투스를 통한 실시간 데이터 수신
- 자동 세션 기록
- 실시간 성과 트래킹

### 3.2 스윙 분석 시스템
#### 3.2.1 비디오 촬영
- 다각도 촬영 지원 (정면, 측면, 상단)
- 촬영 가이드라인 제공
- 슬로우모션 재생
- 프레임별 분석

#### 3.2.2 AI 기반 자동 분석
- **분석 요소**
  - 스탠스 자세
  - 백스윙 길이
  - 임팩트 각도
  - 팔로우스루
  - 헤드 움직임
  - 어깨 정렬

- **피드백 제공**
  - 개선점 자동 감지
  - 시각적 오버레이
  - 비교 분석 (프로 vs 사용자)
  - 맞춤형 개선 팁

### 3.3 퍼팅 템포 트레이너
#### 3.3.1 스트로크 패턴 연습
- **4가지 기본 패턴**
  1. 짧고 짧게 (Short-Short)
  2. 짧고 길게 (Short-Long)
  3. 길고 짧게 (Long-Short)
  4. 길고 길게 (Long-Long)

#### 3.3.2 시청각 가이드
- **시각적 요소**
  - 움직이는 진행 바
  - 색상 변화 (백스윙: 파랑, 다운스윙: 초록)
  - 템포 그래프 실시간 표시

- **청각적 요소**
  - 비트음 제공
  - 사용자 맞춤 BPM 설정
  - 음성 가이드 옵션

#### 3.3.3 개인화 설정
- 사용자별 선호 템포 저장
- 심리 상태별 템포 추천
- 거리별 템포 프리셋

### 3.4 온라인 대전 및 커뮤니티

#### 3.4.1 매칭 시스템
- **매칭 옵션**
  - 실력 기반 매칭
  - 평균 점수 기반 매칭
  - 랜덤 매칭
  - 친구 대전

- **게임 모드**
  - 비동기 대전 (Phase 1)
  - 실시간 대전 (Phase 2)
  - 일일 챌린지
  - 주간 토너먼트

#### 3.4.2 소셜 기능
- 친구 추가/관리
- 프로필 커스터마이징
- 연습 기록 공유
- 댓글 및 응원 기능

#### 3.4.3 랭킹 시스템
- **리더보드**
  - 전체 랭킹
  - 친구 랭킹
  - 지역별 랭킹
  - 게임 모드별 랭킹

- **티어 시스템**
  - 브론즈 → 실버 → 골드 → 플래티넘 → 다이아몬드
  - 시즌제 운영
  - 보상 시스템

### 3.5 통계 및 분석

#### 3.5.1 개인 통계
- 거리별 성공률
- 일/주/월별 연습량
- 실력 향상 그래프
- 약점 분석
- 최적 템포 발견

#### 3.5.2 비교 분석
- 또래 평균과 비교
- 프로 데이터와 비교
- 과거 자신과 비교

## 4. UI/UX 디자인 원칙

### 4.1 디자인 컨셉
- **미니멀리즘**: 깔끔하고 집중력 있는 인터페이스
- **골프장 느낌**: 그린 색상 기반, 잔디 텍스처 활용
- **데이터 시각화**: 직관적인 차트와 그래프

### 4.2 주요 화면 구성
1. **홈 화면**
   - 오늘의 연습 목표
   - 빠른 시작 버튼
   - 최근 기록 요약

2. **연습 모드**
   - 실시간 입력 인터페이스
   - 세션 통계
   - 즉각적 피드백

3. **분석 화면**
   - 스윙 영상 분석
   - 통계 대시보드
   - 개선 제안

4. **커뮤니티**
   - 대전 매칭
   - 리더보드
   - 친구 활동

### 4.3 사용자 경험 최적화
- 원터치 데이터 입력
- 제스처 기반 네비게이션
- 다크 모드 지원
- 접근성 고려 (글자 크기, 색상 대비)

## 5. 기술 스택

### 5.1 프론트엔드
- **플랫폼**: React Native (Android + iOS 크로스플랫폼)
- **상태관리**: Redux Toolkit 또는 Zustand
- **비디오 처리**: react-native-video, FFmpeg
- **차트/그래프**: Victory Native 또는 react-native-chart-kit

### 5.2 백엔드
- **서버**: Node.js + Express
- **데이터베이스**: PostgreSQL (유저 데이터) + Redis (세션/캐시)
- **실시간 통신**: WebSocket (Socket.io)
- **비디오 저장**: Firebase Storage

### 5.3 AI/ML
- **스윙 분석**: TensorFlow Lite (Android) / Core ML (iOS)
- **포즈 감지**: MediaPipe
- **클라우드 AI**: Google Cloud Vision API

### 5.4 하드웨어 연동 (Phase 2)
- **블루투스**: BLE (Bluetooth Low Energy)
- **프로토콜**: GATT 프로파일
- **데이터 포맷**: JSON
